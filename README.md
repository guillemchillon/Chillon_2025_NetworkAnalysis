# A Network Study of Gene Expression Across Human Brain Development

This project analyzes the transcriptomic network architecture of the developing human brain using the BrainSpan atlas.

## Project Structure
* **`code.qmd`**: Main analysis script (R & Python).
* **`data/`**: Contains source files.
* **`results/`**: Generated figures.
* **`Project_3.pdf`**: Final manuscript report.
